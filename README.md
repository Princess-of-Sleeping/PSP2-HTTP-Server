PoC?<br>

<br>
work in vita3k<br>
<br>
## Access processing example
http://host:8080/ \-> ux0:www/index.html<br>
http://host:8080/text.html \-> ux0:www/text.html<br>
http://host:8080/image/img1.png \-> ux0:www/image/img1.png<br>
